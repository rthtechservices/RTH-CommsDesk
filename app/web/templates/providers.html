Provider tokens and credentials are managed outside the UI. To reconfigure:

  • Gmail: re-run the OAuth flow via python -m app.connectors.gmail.oauth_setup
  • Microsoft Graph: re-run via python -m app.connectors.microsoft_graph.oauth_setup
  • AI provider: set the OPENAI_API_KEY or equivalent environment variable
