Configuration guidance

Provider tokens and credentials are managed outside the UI. This page observes configuration only. No credentials are modified here.

  • Gmail: run .\scripts\reauth-commsdesk.ps1 -Gmail. Required scopes: https://www.googleapis.com/auth/gmail.readonly, https://www.googleapis.com/auth/gmail.compose, https://www.googleapis.com/auth/gmail.send, https://www.googleapis.com/auth/gmail.modify.
  • Google Calendar: run .\scripts\reauth-commsdesk.ps1 -GoogleCalendar. Required scopes: https://www.googleapis.com/auth/calendar.freebusy, https://www.googleapis.com/auth/calendar.events.
  • Microsoft Graph: run .\scripts\reauth-commsdesk.ps1 -MicrosoftGraph. Required scopes: User.Read Mail.Read offline_access.
  • AI provider: set the OPENAI_API_KEY or equivalent environment variable
